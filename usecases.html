Use cases

The use cases for Bird are definition and parsing of any binary data format for:

• security analysis
• fraud detection
• internet traffic analysis
• forensics research

The use cases for Nescio are definition and implementation of anonymization rules as needed in:

• privacy protection
• publishing large data sets in a wide range of domains
• publishing data for detecting DDOS attacks
• exchange of data between financial service providers
• exchange of medical data between health service providers
